# Gunakarthik Naidu Lanka Portfolio

Professional software engineering portfolio built with Next.js, React, Tailwind CSS, and Framer Motion.

## About

This portfolio highlights:
- Backend and full-stack engineering focus
- Professional experience (Velocity Tech, EPICS, ASU Teaching Assistant)
- Projects (AI Meeting Assistant, Foster Arizona Platform, Hospital DBMS)
- Education (ASU accelerated 4+1 Computer Science track with Business minor)

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run start
```

## Project Structure

- `src/app` - App router pages
- `src/pages` - Main page content components
- `src/components` - Shared UI components
- `src/config` - Contact/profile configuration
- `public` - Images and downloadable assets
